# GenAI Technical Assistant UI

## Overview
Simple frontend interface for interacting with the RAG-based technical assistant.

## Features
- User query input
- Answer display
- Retrieved source chunks with metadata

## How it works
Frontend sends query to backend API → receives answer + context → displays results.

## How to run
Just open `index.html` in browser.

## Notes
Designed for POC; can be extended to production UI (React / API integration).
